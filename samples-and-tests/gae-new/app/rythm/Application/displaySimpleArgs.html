@extends(main, title:"simple template arguments") @args String country, String capital This sample shows how to pass simple argurments to your templates.

Template code

@verbatim{ 
	@extends(main, title:"simple template arguments")
	@args String country, String capital


	@capital is the capital of @country
}

Action code

@verbatim{ 
	public static void displaySimpleArgs() {
		renderArgs.put("country", "France");
		renderArgs.put("capital", "Paris"); 


		render();
	}
}

Output

@capital is the capital of @country
